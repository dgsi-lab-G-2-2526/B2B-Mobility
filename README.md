# B2B-Mobility
En este repositorio vamos a realizar la elaboración de las practicas de la asignatura "Desarrollo y Gestión de Sistemas de Información

## **Sprint Review - Sprint 1 (Fundamentos y Entorno)**

Durante el Sprint 1, el equipo B2B-Mobility sentó las bases del proyecto de Carsharing corporativo. Logramos definir nuestra propuesta de valor B2B2C, identificar a los stakeholders clave y delimitar el proceso E2E central (Ciclo Integral de Reserva, Uso y Adecuación del Vehículo). Además, configuramos con éxito nuestro entorno de trabajo ágil en GitHub (repositorio, tablero Kanban y Definition of Done) y diseñamos el artefacto de visión inicial del SIA bajo una arquitectura cliente/servidor, cumpliendo con todos los objetivos iniciales de trazabilidad.

## **Sprint Review - Sprint 2 (Competitividad y Estrategia)**

En este Sprint 2, hemos evolucionado nuestro modelo de negocio respondiendo a las presiones competitivas del mercado mediante estrategias claras. El equipo logró modelar el proceso actual (As-Is) para detectar cuellos de botella y diseñó un flujo optimizado (To-Be) introduciendo quick wins, respaldado por 3 KPIs con sus respectivas fuentes y baselines. Asimismo, aseguramos la gobernanza del proyecto identificando riesgos críticos operativos y de compliance, estableciendo una métrica de calidad del dato (DQ) , y documentando un blueprint tecnológico conceptual alineado con sistemas corporativos (CRM/ERP/SCM). Todas las tareas del backlog se han completado y enlazado correctamente asegurando la trazabilidad exigida.
